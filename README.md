# Игры разума


Набор простых игр

- Проверка на чётность (Even)
- Калькулятор (Calc)
- Наибольший общий делитель (GCD)
- Арифметическая прогрессия (Progression)
- Простое ли число? (Prime)

Для запуска выполнить в директории проекта `app`:  
```sh
make install  
make run-dist
```
